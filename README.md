## Development

To view this presentation locally, first run `npm install`.

### Build
Outputs HTML

``` sh
npm run html
```

### Watch
Watches for changes, builds html

``` sh
npm run watch
```

### Server
Serves project at `http://localhost:8080`

``` sh
npm run server
```
